XbimMultiValueDictionaryTKey, TValueCreateTValueCollection Method (IEnumerableKeyValuePairTKey, ICollectionTValue, IEqualityComparerTKey)

Initializes a new instance of the XbimMultiValueDictionaryTKey, TValue class that contains elements copied from the specified IEnumerable<KeyValuePair<TKey, ICollection<TValue>>> and uses the specified IEqualityComparerT for the TKey type. The internal dictionary will use instances of the TValueCollection class as its collection type.
