XbimXmlWriter4
Constructor of the XML writer. This writer implements conventions for IFC4 XML and needs to get configuration with speficic cases like inversed inverses and other special cases. Default settings and configuration are for IFC4 Add1. If you pass configuration and settings make sure both are for the same schema definition.
