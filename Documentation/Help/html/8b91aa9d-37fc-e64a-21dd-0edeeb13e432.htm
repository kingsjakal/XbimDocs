JSONWritter
You can use this writer to create JSON form of EXPRESS defined data. It implements abstract XmlWriter so you can pass it as a writer to XbimXmlWriter4. As a result you will get JSON instead of XML. Result only contains text fields even for numeric values because of the way XML work. This might be extended in the future.
